Choose a policy to use for your scan. Policies are located in "policies" directory which is located in the directory defined above or in the default directory used by ZAP.
If no policies are availables, ZAP will use the default policy.
In order to load the polocies list, the job must be built once in order to create the workspace.
So do as follows :
1-create your job
2-Fill the ZAProxy default directory text box.
3-Save the job and build it (the buil will faillure, it's normal)
4-Reload the page
5-Now you can have access to the policies list and fill all required fields for the scan
